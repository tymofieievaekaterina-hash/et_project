# Определение переодов очередей на кассах
Расчет выполнен в MS Excel и MS SQL Server.
Как источник данных использована объединенная таблица из файлов "store_checkout_queues" и "store_stores". Объединение данных по чекам и магазинам, фильтрация данных по магазинам с id=98451680 и id=12864064 произведидино в Excel с помощью функционала надстройки Excel - Power Query. В результате получена таблицу со следующими полями:
checks_number
store_id
employees_id
quantity
selling_price
checkout_id
start_operation_dt
end_operation_dt

Такая же логика объединения и выборки магазинов воспроизведена в виде SQL запроса.

Логика расчета:
Операция считается находящейся в очереди, если время между текущей операцией и предыдущей менеше или ровно минимальному времени на выполнение одной операции конкретного кассира (учитывая количество позиций в чеке ) предидущей операции. Для расчета берем минимальное время на операцию кассира (выполняющего данную операцию) с таким же количеством товара как в предидущей операции, так как время выполнения операций зависит от количество позиций в ней.

Этапы расчета:
1. Агрегируем данные для расчета.
2. Определяем минимальное времени на операцию (в секундах) в зависимости от количества товаров в чеке для каждого кассира.
3. Берем только те операции, которые по логике считаем как те, что в очереди.
4. Определяем количество операций в очереди в час.
5. Выводим результирующуб таблицу в виде таблицы с полями: период, колличество операций в очереди

	На основании проведенного расчета можно сделать следующие выводы о распределении очередей по часам:	
	1. Пиковые часы: Очереди на кассах наиболее длинные в период с 11:00 до 15:00 и с 17:00 до 21:00. В эти часы количество операций в очереди достигает максимальных значений, что указывает на высокую загруженность касс.
	2. Минимальные очереди: Наименьшее количество операций в очереди наблюдается в ночные и утренние часы (с 00:00 до 08:00) и в поздние вечерние часы (после 22:00). В эти периоды очереди практически отсутствуют или минимальны.
		
    Таким образом, периоды с наибольшей загруженностью касс и длинными очередями приходятся на обеденное время (12:00-15:00) и вечерние часы (18:00-21:00) с максимальными значениями с 12:00 до 13:00,  с 17:00 до 18:00 и  с 20:00 до 21:00. В остальное время очереди минимальны или отсутствуют.
